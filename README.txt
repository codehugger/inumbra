Installation instructions for Inumbra - journey through the dark

The game can be run using your keyboard and mouse or handheld controllers.
Follow these instructions to run the game..

MAC OSX:
1. unzip the zip file using your unwrapper of choice.
2. run file
optional:
3. plug in your Playstation 4 controller

WINDOWS:
1. unwrap the compressed folder using your unwrapper of choice.
2. run exe file
optional:
3. plug in your XBOX ONE controller
